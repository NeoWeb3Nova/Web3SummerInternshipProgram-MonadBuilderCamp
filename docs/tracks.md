# 三条 Builder Track

## Tech Track: 技术与产品实现方向

### 适合谁
- 想成为 Web3 Frontend Engineer、DApp Engineer、Solidity Intern、DevRel Engineer
- 有编程基础，希望学会链上产品开发

### 核心技能树
- 钱包连接（MetaMask / WalletConnect / RainbowKit）
- 智能合约开发（Solidity）
- 合约框架（Foundry / Hardhat）
- 前端交互（viem / ethers.js / wagmi）
- 事件监听与索引
- 测试与安全基础
- README 与技术文档写作

### 建议学习路径
1. Solidity 基础语法
2. Foundry 环境搭建与测试
3. 在 Monad Testnet 部署合约
4. 用 Vite + React + wagmi 搭建前端
5. 连接钱包并读取/写入合约
6. 写测试、写 README、部署文档

### 代表产出
- 可运行的 DApp Demo
- 合约地址与部署脚本
- GitHub Repo
- 部署文档
- 技术复盘

## Ops Track: 运营、增长与生态协作方向

### 适合谁
- 想进入 Community、Growth、Marketing、Event Ops、BD、Ecosystem 方向
- 对人和传播敏感，喜欢组织活动与内容

### 核心技能树
- 生态调研与用户观察
- Meme Campaign 设计
- 活动 SOP 编写
- 社区反馈机制
- 增长复盘
- Demo Day 执行

### 建议实践路径
1. 选择 Monad 生态中的 3-5 个项目进行桌面研究
2. 加入相关社区，观察用户讨论与痛点
3. 设计一个 Meme / 内容 / 活动 Campaign
4. 执行最小版本并收集反馈
5. 记录数据与增长复盘

### 代表产出
- Campaign 方案
- 传播数据截图
- 活动 SOP
- 用户反馈表
- 社区增长复盘
- Demo Day 执行记录

## Research Track: 研究、分析与叙事方向

### 适合谁
- 想成为 Ecosystem Researcher、Protocol Research Intern、Product Researcher、Technical Writer
- 喜欢阅读、分析、写作和讲故事

### 核心技能树
- 架构解释（把复杂协议讲清楚）
- 生态地图绘制
- 项目分析框架
- 标准 / 治理笔记
- 产品适配论证
- 研究写作

### 建议研究路径
1. 阅读 Monad 白皮书/文档
2. 建立项目分析框架
3. 选择 2-3 个 Monad 生态项目做对比
4. 写一篇分析文章或生态地图
5. 制作可用于公开展示的材料

### 代表产出
- 研究文章
- Product Thesis
- 项目分析
- 生态地图
- 公开演讲/展示材料
