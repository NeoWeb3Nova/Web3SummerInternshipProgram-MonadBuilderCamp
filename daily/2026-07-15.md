## 今日收获

1. 对链上整数参数，类型是 `number` 不代表值仍然安全；必须在 `BigInt` 转换前检查 `Number.isSafeInteger`。
2. TDD 的价值不只是补覆盖率，而是先证明测试确实能抓住原始缺陷。
3. 小 PR 也需要 changeset、完整质量门、整洁 commit 和明确非范围。
4. 上游 Draft PR #31 已把新框架改成 string-only integer schema；当前 PR 可能被 supersede，但应该透明披露，而不是隐瞒合并风险。
5. GitHub 的 `action_required` 且 jobs 为 0，代表工作流还没运行，不能当成 CI 失败或通过。
