# Week 2 Ops Track: 运营、增长与生态协作

## 本周目标

完成 Monad 生态调研与用户观察，设计并执行一个最小化增长实验。

## Day-by-Day 任务

### Day 8: 生态地图
- [ ] 识别 Monad 生态中的 5 个代表项目
- [ ] 记录每个项目的：名称、定位、Twitter、Discord、网站
- [ ] 在 `submissions/week-02/` 创建 `ecosystem-map.md`

### Day 9: 用户观察
- [ ] 加入 2-3 个 Monad / Web3 相关社群
- [ ] 记录 10+ 条用户讨论 / 痛点 / 问题
- [ ] 整理观察到 `user-insights.md`

### Day 10: Campaign 设计
- [ ] 选择一个增长目标（如：提升 Monad Testnet 活跃度、推广某个工具）
- [ ] 设计 Campaign 主题与 Hook
- [ ] 制定内容日历与渠道

### Day 11: 内容制作
- [ ] 制作 3 条以上社交媒体内容
- [ ] 使用 AI 协助撰写，标注 AI 辅助部分
- [ ] 保存草稿到 `assets/week-02/`

### Day 12: 最小实验
- [ ] 在至少 1 个渠道发布内容
- [ ] 记录发布时间、平台、互动数据
- [ ] 收集最早的用户反馈

### Day 13: 反馈与迭代
- [ ] 分析数据：曝光、互动、转发、评论
- [ ] 调整下一轮内容
- [ ] 更新 Campaign 方案

### Day 14: 复盘与 Week 2 提交
- [ ] 整理增长数据
- [ ] 撰写复盘报告
- [ ] 提交到 `submissions/week-02/`
