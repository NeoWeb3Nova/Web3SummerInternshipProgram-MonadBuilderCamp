# Week 2 Research Track: 研究、分析与叙事

## 本周目标

完成 Monad 技术或生态的深度研究，输出一篇可公开展示的分析材料。

## Day-by-Day 任务

### Day 8: 文献阅读
- [ ] 阅读 Monad 白皮书 / 技术文档
- [ ] 阅读 Monad 官方 blog / 技术 thread
- [ ] 整理关键术语与概念

### Day 9: 框架建立
- [ ] 确定研究问题
- [ ] 选择分析框架（如：技术架构 / 竞品分析 / 产品适配）
- [ ] 在 `submissions/week-02/` 创建 `research-outline.md`

### Day 10: 数据收集
- [ ] 搜集 3-5 个相关项目/案例
- [ ] 记录每个案例的关键数据与结论
- [ ] 使用 AI 协助整理，标注使用范围

### Day 11: 分析与论证
- [ ] 使用框架分析案例
- [ ] 形成自己的观点
- [ ] 寻找支持/反驳论点

### Day 12: 草稿撰写
- [ ] 撰写研究文章初稿
- [ ] 包含：引言、方法、分析、结论
- [ ] 使用 AI 辅助修饰，但确保观点自己掌握

### Day 13: 反馈与完善
- [ ] 发给 1-2 人审阅
- [ ] 收集建议
- [ ] 修改并完善

### Day 14: 复盘与 Week 2 提交
- [ ] 最终定稿
- [ ] 制作可视化材料（可选：生态地图、表格、图表）
- [ ] 提交到 `submissions/week-02/`
