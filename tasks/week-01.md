# Week 1: 认知奠基期 / 进入 Onchain World

## 本周目标

从零开始进入 Web3，建立共同基础，完成第一次链上体验，并把学习过程记录成可提交材料。

开营仪式关键词：钱包创建、链上体验、产品认知、AI 工具入门、每日打卡、19:00 嘉宾分享、20:00 co-learning。

## 每日固定动作

- [ ] 查看学习平台日历和当天任务
- [ ] 参加或回看 19:00 嘉宾分享
- [ ] 参加或回看 20:00 co-learning / 助教答疑
- [ ] 完成学习平台打卡
- [ ] 更新 `daily/YYYY-MM-DD.md`
- [ ] 记录 AI 帮了什么、自己做了什么判断

## Day-by-Day 任务

### Day 1: 审核、平台、助教

- [ ] 确认已通过审核
- [ ] 登录学习平台，查看日历和任务表
- [ ] 选择助教（Cuder / 书生 / 尤拉 / 花花 / Beam）
- [ ] 加入官方群/频道，确认助教备注
- [ ] 阅读 `docs/opening-ceremony.md`
- [ ] 在 `submissions/week-01/onboarding.md` 记录：选择的助教、选择原因、主攻方向初判

### Day 2: 钱包与安全

- [ ] 安装 MetaMask 或兼容 EVM 的钱包
- [ ] 创建专用测试钱包，不使用主资产钱包
- [ ] 纸笔备份助记词，不上传云端，不发给 AI
- [ ] 记录钱包地址到 `submissions/week-01/wallet.md`
- [ ] 写 300 字：钱包、私钥、助记词、地址之间的关系

### Day 3: Monad Testnet 配置

- [ ] 获取 Monad Testnet RPC / Chain ID / Explorer 信息
- [ ] 在钱包中添加 Monad Testnet
- [ ] 从 Faucet 领取测试网代币
- [ ] 记录网络配置与 Faucet 链接到 `submissions/week-01/network-config.md`
- [ ] 截图保存到 `assets/week-01/`

### Day 4: 第一笔链上交易

- [ ] 完成第一笔测试网转账或合约交互
- [ ] 记录交易哈希和 Explorer 链接
- [ ] 截图保存到 `assets/week-01/`
- [ ] 在 `submissions/week-01/first-tx.md` 写清楚：我做了什么、为什么成功、遇到什么问题

### Day 5: AI 工具与 Vibe Coding 入门

- [ ] 配置至少一个 AI coding / learning 工具
- [ ] 用 AI 解释一个 Web3 概念：MPC 钱包、EIP、Gas、EVM、RPC 任选其一
- [ ] 用 AI 生成一个简单 DApp / 合约想法列表
- [ ] 在 `submissions/week-01/ai-learning.md` 记录 prompt、输出、人类判断和修正

### Day 6: Monad 技术基础

- [ ] 阅读 `docs/monad-technical-notes.md`
- [ ] 用自己的话解释：为什么需要高吞吐链
- [ ] 画出 Monad 三个执行层创新：流水线化共识与执行、乐观并行执行、MonadDB
- [ ] 在 `submissions/week-01/monad-notes.md` 输出 500-800 字学习笔记

### Day 7: 周复盘与提交

- [ ] 整理 Week 1 所有提交物
- [ ] 检查每日打卡是否齐全
- [ ] 写 `submissions/week-01/retro.md`
- [ ] 更新 README 中的进度（如有）
- [ ] 准备 Week 2 主线选择：Tech / Ops / Research
