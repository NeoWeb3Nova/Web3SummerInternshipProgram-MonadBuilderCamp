# Week 1: 进入 Onchain World

## 本周目标

建立共同基础，完成第一次链上交互。

## Day-by-Day 任务

### Day 1: 报名与环境准备
- [ ] 确认已在 web3career.build 完成报名
- [ ] 加入社群/频道（如有）
- [ ] 安装 MetaMask 或兼容 Monad 的钱包
- [ ] 创建本项目目录并 clone 到本地

### Day 2: 钱包配置
- [ ] 创建新钱包或使用专用测试钱包
- [ ] 备份助记词（纸笔，多地保存）
- [ ] 记录钱包地址到 `submissions/week-01/wallet.md`

### Day 3: Monad 测试网配置
- [ ] 获取 Monad Testnet RPC 信息
- [ ] 在钱包中添加 Monad Testnet
- [ ] 从 Faucet 领取测试网代币
- [ ] 记录网络配置与 Faucet 链接

### Day 4: 第一次交易
- [ ] 完成第一笔测试网转账
- [ ] 记录交易哈希
- [ ] 截图保存到 `assets/week-01/`

### Day 5: 开发环境准备
- [ ] 安装 Node.js + npm/pnpm
- [ ] 安装 Foundry (`curl -L https://foundry.paradigm.xyz | bash`)
- [ ] 验证 `forge --version`
- [ ] 安装 Git 并配置用户名/邮箱

### Day 6: 第一个合约
- [ ] 使用 Foundry 创建项目: `forge init`
- [ ] 编写 `HelloWorld.sol`
- [ ] 在 Monad Testnet 部署
- [ ] 记录合约地址

### Day 7: 复盘与 Week 1 提交
- [ ] 填写本周学习日志
- [ ] 整理 Week 1 提交物到 `submissions/week-01/`
- [ ] 审查是否达成本周目标
- [ ] 更新 README 中的进度
