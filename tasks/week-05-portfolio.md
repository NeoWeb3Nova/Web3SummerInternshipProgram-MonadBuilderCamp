# Week 5: Resume & Portfolio Workshop

## 本周目标

把前 4 周的学习、项目、反馈和 Hackathon 结果整理成可展示的职业证据：作品集、简历 bullet、面试讲稿、个人品牌内容和后续机会清单。

开营仪式提到，第 5 周重点包括：作品集整理、简历优化、面试准备、个人品牌建设、生态连接和后续实习 / 生态大使机会。

## 本周原则

- 不只说“我学了”，要展示“我做了什么、产生什么证据”
- 每个成果都尽量链接到 Repo、Demo、交易哈希、文章、截图或反馈
- 简历表达要具体、可验证、有数据
- AI 协作要透明：AI 做了什么，人判断了什么
- 后续机会要主动跟进，不等别人来找

## 简历 Bullet 模板

```text
- [动作] + [对象] + [方法/工具] + [结果] + [可验证证据]
- 示例: 在 Monad Testnet 上设计并部署了一个 X 合约 DApp，使用 Foundry + React + viem 完成钱包交互和链上写入，提交 Y 个交易哈希并在 Demo Day 完成展示。
```

## 面试常见问题

- 为什么选择 Monad？
- 你在这个项目中做了什么？
- 这个项目解决了什么真实问题？
- AI 帮了你什么？你做了什么判断？
- 遇到了什么困难？如何解决？
- 如果再做一次，会怎么改进？
- 这个项目下一步如何成为真实产品？
