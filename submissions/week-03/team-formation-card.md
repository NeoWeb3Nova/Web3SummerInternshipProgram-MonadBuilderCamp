## 3. 共同目标（本周）

### 必须做到（Definition of Done · Mini Demo）

- [ ] 有清晰用户与痛点（Problem & User Card）  
- [ ] 只保留 **一条** 端到端链路，砍掉其余  
- [ ] 链路可演示：意图输入 → Plan / simulate 结果 → 确认态  
- [ ] 至少一类 **Demo Evidence**：合约地址 / 交易哈希 / 可运行 Repo / 录屏 之一（优先多类）  
- [ ] 3 分钟 Pitch 能讲完：问题 → 方案 → 演示 → 为什么 Monad → 下一步  
- [ ] 至少 1–3 人试用或看过 Demo 并留下反馈  

### 明确不做什么（本周）

- 不做通用多协议 Agent  
- 不做自动执行 / 托管私钥 / 无人值守策略  
- 不做多链  
- 不追求完整产品与增长漏斗
