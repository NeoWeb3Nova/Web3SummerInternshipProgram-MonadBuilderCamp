# 第一个 Monad 合约部署实验

## 实验目标

使用 Foundry 部署一个简单合约到 Monad Testnet。

## 环境

- Foundry
- Monad Testnet

## 实验步骤

1. `forge init`
2. 编写 `HelloWorld.sol`
3. 编译 `forge build`
4. 配置私钥环境变量
5. 部署 `forge script`
6. 验证合约
